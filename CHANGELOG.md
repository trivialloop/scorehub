# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0]

### Added
- Add wingspan game

### Changed
- Improve game selection

## [1.2.0]

### Added
- Allow to use custom color for the player

### Changed
- Improve readibility for games
- Improve popup end of games

## [1.1.0]

### Added
- Add Skyjo game

## [1.0.1]

### Fixed
- Fix dark mode

## [1.0.0]

### Added
- Initial release
- Yahtzee game support with full scorecard
- Player management system with custom colors
- Comprehensive statistics tracking
  - Win/loss/draw percentages
  - Best and worst scores per player
  - Player rankings
- Top 20 high scores leaderboard
- Multi-language support (English and French)
- Game history persistence

### Features
- Support for 1+ players per game
- Color-coded player identification
- Automatic player rotation during games
- Statistics screen with detailed player performance
- General statistics with best player and score highlights
