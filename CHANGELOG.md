## [1.0.0] - 2026-02-03

### Added
- Initial release
- Yahtzee (Yams) game support with full scorecard
- Player management system with custom colors
- Comprehensive statistics tracking
  - Win/loss/draw percentages
  - Best and worst scores per player
  - Player rankings
- Top 20 high scores leaderboard
- Multi-language support (English and French)
- Game history persistence

### Features
- Support for 1+ players per game
- Color-coded player identification
- Automatic player rotation during games
- Statistics screen with detailed player performance
- General statistics with best player and score highlights
